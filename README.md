# grid_server
    A basic server and client in python

## Installation

1. 
```sh
git clone https://github.com/leanke/grid_server.git
```

2. 
```sh
pip install -e ./grid_server
```

### Running
start the server
```sh
python run.py --mode server
```
or to run the client
```sh
python run.py --mode client
```
